Special Notes

MS5 does not work with Visual Studio Express Edition. This is because the Express Editions do not allow Visual Studio extensibility. We have plans in the near future to provide some basic support to Express users.

New Features

  • Integrated Debugger
  • Custom Project type

Known Issues

  • Stepping will cause a crash of the guest.
  • Only QEMU and VMWare Workstation and Player are supported. Others will be resupported soon, but can be used manually by mounting the ISO. ISO however isnt supported in this release, so you need to run the VMWare version and pluck the ISO from the output directory
