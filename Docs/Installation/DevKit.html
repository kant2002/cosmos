Prerequisites

  • Visual Studio 2008 Professional or higher
    • Express Editions will be supported again in the near future
    • Visual Studio 2010 is not currently supported
  • Team Foundation Client
    • This can be found on your Visual Studio 2008 installation media under TFCClient folder. It is best to install this before installing Visual Studio 2008 SP1. If you do not wish to use the integrated source control, you can use the SVN bridge instead.
  • Visual Studio 2008 Service Pack 1
  • Visual Studio Team System 2008 Service Pack 1 Forward Compatibility Update for Team Foundation Server 2010
    • This is needed if you plan to use the integrated source control in Visual Studio
  • InnoSetup QuickStart Kit
    • This is required to build the setup kit which is used to build and instal the Visual Studio integration libaries for Cosmos
    • During install it will ask you about optional components to install. Leave them all checked.

Installation

  • Get source from CodePlex.
  • Run /build/VSIP/install.bat
  • Run /build/VSIP/install.bat AGAIN
