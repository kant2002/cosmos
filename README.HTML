Cosmos Source Structure

'\'

At first the Cosmos Source can look like a daunting thing. This folder structure guide should help you to find what you're looking for.

Build

Tools & 3rd party applications/files required for the building and running of a Cosmos OS

BuildProcessTemplates

A folder added by Codeplex for some reason...

Docs

Various user end documentation for the Cosmos User Kit. Quite badly structured.

QA

A single build script for doing something or other

Resources

Dependency .dll's that Cosmos needs

Setup2

Files needed for the installer including the inno setup file

source

The first folder full of source and an assortment of used and non-used code

source2

Lots of folders with source in to explain, a seperate page is linked above.

TeamProjectConfig

A single TeamProject.config file

XSharp

Things for XSharp (the cosmos compiler language) but not all the source
